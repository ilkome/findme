Gulp - это таск-менеджер: приложение, которое используется для автоматизации разработки проекта: 
	- автоматически собирает из исходников готовый сайт html+css
	- создает виртуальный сервер и при изменении файлов с версткой (Stylus, Jade, JS) обновляет его
Подробнее в gulpfile.js



// УСТАНОВКА ПРОЕКТА Gulp + Jade + Stylus
// = = = = = = = = = = = = = = = = = = = = = = = = //
	1. Установить node.js http://nodejs.org/

	2. В коммандной строке выполнить:
		npm install gulp -g

	3. В корне папки проекта запустить install-gulp.cmd или в ком. строке выполнить:
		npm install gulp gulp-stylus gulp-jade gulp-autoprefixer gulp-connect gulp-plumber gulp-watch

	4. Запустить run-gulp.cmd или в ком. строке выполнить:
		gulp

	5. Верстка будет доступна по адресу
		http://localhost:8080



// СТРУКТУРА
// = = = = = = = = = = = = = = = = = = = = = = = = //
jade ............... jade шаблоны (препроцессор HTML)
node_modules ....... системная папка с модулями node.js
site ............... скомпилированная верстка
styl ............... stylus шаблоны (препроцессор CSS)
gulpfile.js ........ настройки Gulp
gulp-install.cmd ... установка gulp в текущую папку
gulp-run.cmd........ запуск Gulp
info.txt ........... информация



// ВОЗМОЖНЫЕ ОШИБКИ:
// = = = = = = = = = = = = = = = = = = = = = = = = //
	1. При открытии install-gulp.cmd или run-gulp.cmd консоль открывается и закрывается:
		Открыть консоль в папке проекта и ввести команду.

	2. Нельзя использовать "[]" в названии папок в пути к проекту.




// ПРИНЦИП НАПИСАНИЯ СТИЛЕЙ:
// = = = = = = = = = = = = = = = = = = = = = = = = //
1. Названия основных классов отсортированно по алфавиту. шапка(header) - всегда наверху, подвал(footer) - внизу
2. Стили пишуться по сл. порядку:
	display, z-index
	position
	width, height
	margin, padding
	color, font
	background
	border
